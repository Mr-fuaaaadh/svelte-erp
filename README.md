# AIPSA ERP - Enterprise Management Platform

A premium, high-performance School Management System (SMS) and Enterprise Resource Planning (ERP) platform built with **SvelteKit 5**, **Tailwind CSS 4**, and the **Internet Computer Protocol (ICP)**.

## Project Overview

AIPSA ERP is designed to provide a seamless, high-fidelity experience for school administrators, teachers, parents, and students. It features a modern glassmorphism design, real-time analytics, and secure blockchain-backed infrastructure.

### Key Features
- **Modern Dashboard**: High-fidelity overview of school metrics.
- **School Management**: Comprehensive management of schools across India.
- **Student & Teacher Portals**: Dedicated interfaces for key stakeholders.
- **Secure Infrastructure**: Powered by ICP for decentralized and secure operations.

## Technology Stack
- **Frontend**: SvelteKit 5 (Runes architecture)
- **Styling**: Tailwind CSS 4 (OKLCH color system)
- **Icons**: Lucide Svelte
- **Infrastructure**: Internet Computer Protocol (ICP)
- **State Management**: Svelte Runes

## Getting Started

### Prerequisites
- Node.js >= 16.0.0
- pnpm

### Installation
```bash
pnpm install
```

### Development
```bash
cd src/frontend
pnpm dev
```

### Production Build
```bash
pnpm build
```

## Architecture

- `src/frontend`: SvelteKit application logic and UI.
- `src/backend`: Motoko canisters for the Internet Computer.
- `src/lib`: Shared utilities and stores.
